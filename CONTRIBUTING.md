## Welcome

OpenFlow is a living governance framework. We welcome contributors who share our core axiom:

> **DNA + AI = Complete** — human judgment and AI capability are partners, never replacements.

Before contributing, please read this document fully.

---

## Zero-Authority Principle

No single contributor — including founders — can override OpenFlow's ethical safeguards.  
All changes to core governance logic require at minimum **one founder review + one community review**.

---

## Open Cut Disclosure (Required)

Every Pull Request that touches governance logic, protocols, or licensing **must** include an Open Cut declaration in the PR description:

```
## Open Cut Declaration
- [ ] I have paused and reviewed the ethical implications of this change
- [ ] I have considered potential harm vectors
- [ ] I disclose any conflict of interest
- [ ] I defer to the OpenFlow Board if this change affects Zero-Authority axioms
```

PRs missing this declaration on governance files will be closed without review.

## How to Submit a PR

1. Fork the repo
2. Create a branch: `feature/your-change` or `fix/your-fix`
3. Make your changes
4. Add Open Cut Declaration to PR (if applicable)
5. Submit PR — a founder will review within 7 days

---

## Code of Conduct

All contributors must:
- Respect human dignity in all proposals
- Disclose AI-generated content in contributions
- Not submit changes designed to weaken ethical safeguards

Violations result in immediate removal and repo ban.
